# Data Analysis Projects

Three beginner-level data analysis projects built with Python, SQL, pandas, and matplotlib.

## Projects

### 1. HR Attrition Analysis
- Tools: Python, SQLite (SQL), pandas, matplotlib
- Analyzed 1,000 employee records to find attrition patterns by department and tenure
- Found Sales new-joiners (< 2 years) had the highest exit rate

### 2. Sales Performance Analysis
- Tools: Python, pandas, matplotlib
- Cleaned and analyzed 2,000 sales transactions across 5 categories and 4 regions
- Built pivot tables and monthly trend charts

### 3. Student Score Tracker
- Tools: Python, pandas, matplotlib
- Tracks marks for 25 students across 5 subjects
- Auto-flags students needing support and generates a class report
